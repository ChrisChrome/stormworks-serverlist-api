This is a simple server list API for Stormworks. The following endpoints are available:
/check?address=IP:PORT
/serverList
/docs
/robots.txt

/serverList takes the following optional parameters:
?uptodate - Filters out servers that are not running the latest version
?outdated - Filters out servers that are running the latest version
?version=v1.10.0 - Filters servers by version
?dlc=1|2|3 - Filters servers by DLC
